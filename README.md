# Punto 506 📍

**Punto 506** es una plataforma digital diseñada para centralizar la información, servicios y comercio de la zona de Pococí. El proyecto busca ser el punto de encuentro definitivo entre los habitantes y la oferta local, impulsando la digitalización de la región.

---

## 🎯 Objetivo del Proyecto
Proveer una herramienta moderna, rápida y exclusiva que permita a los usuarios:
* **Descubrir:** Encontrar negocios y servicios locales fácilmente.
* **Conectar:** Facilitar la comunicación directa entre proveedores y clientes.
* **Destacar:** Ofrecer una vitrina de alta visibilidad para los comercios de Pococí bajo un estándar de calidad "Select".

---

## 🎨 Identidad Visual (Branding)
La identidad de **Punto 506** refleja tecnología, exclusividad y pertenencia local.

### 1. Paleta de Colores
| Color | Hex | Uso Sugerido |
| :--- | :--- | :--- |
| **Verde Esmeralda** | `#10B981` | Color principal de acento y energía. |
| **Dorado Premium** | `#D4AF37` | Punto focal y detalles de lujo. |
| **Gris Carbón** | `#1F2937` | Fondos "Dark Mode" y textos principales. |
| **Blanco Nieve** | `#F9FAF B` | Fondos "Light Mode" y espacios negativos. |

### 2. Tipografía
* **Principal:** `Montserrat` (Bold/Medium). Una fuente geométrica que transmite modernidad y solidez.
* **Secundaria:** `Inter` o `Roboto` (Light/Regular). Utilizada para eslóganes y cuerpos de texto para garantizar legibilidad.

### 3. Elementos Visuales y Logo
* **Concepto:** El logo utiliza una estructura de cruz tipográfica donde la "O" de **PUNTO** sirve como eje vertical para los números **5** (arriba) y **6** (abajo), formando el código 506.
* **El Isotipo:** El símbolo de la columna numérica `5-O-6` puede utilizarse de forma independiente en perfiles de redes sociales, iconos de apps o marcas de agua.
* **Espaciado:** Se recomienda un área de respeto equivalente al tamaño del "punto dorado" alrededor del logo para mantener la limpieza visual.
